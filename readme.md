## Minesweeper

* Fork the repository

* Clone the repository

```bash
git clone https://github.com/<username>/mine-sweeper.git
```

* Go to the directory `mine-sweeper`

```bash
cd mine-sweeper/
```

* Compile the file `main.c` and create the executable `main`

```bash
gcc main.c -o main
```

* Run the game `main`

```bash
./main
```
